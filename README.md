# Policy Modeling

Policy Modeling is a Vite React app with HeroUI and Tailwind CSS already wired up. It is a clean starting point for building the first real policy modeling screen.

## Overview

### Live App

- Production URL: https://policy-modeling.vercel.app
- GitHub repo: https://github.com/indered/policy-modeling

## What's in the App

- The app shows a comparison of the two policies against the user data
- One policy is asset-based, and the other one is a pure LTC policy
- I have tried to build a comparison where we can edit the preferences or data of the user, and the results of the comparison will change based on that
- I made a policy data model based on the type of policy (asset-based or pure LTC), type and range of premiums preferred by the user, inflation rider preferences and the amount of money the customer is paying out of pocket
- There is an age slider too that tells how much the gap policy is covering when LTC starts at that age, taking into consideration inflation too
